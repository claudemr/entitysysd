    Module entitysysd.system

    System management module.

    Classes

    Name Description
    System System abstract class. System classes may derive from it and override prepare, run or unprepare.
    SystemManager Entry point for systems. Allow to register systems.

    Structs

    Name Description
    Order Enum allowing to give special order of a system when registering it to the SystemManager.
    StatEvent Event sent regularly (depending of statRate) to display diverse statistics about the SystemManager.

    Aliases

    Name Type Description
    ISystem System Deprecated. Alias to keep relative backward compatibility with older interface.
