    Interface ISystem

    ISystem interface. System classes must derive from it and implement prepare, run or unprepare.

    Methods

    Name Description
    prepare Prepare any data for the frame before a proper run.
    run Called by the system-manager anytime its method run is called.
    unprepare Unprepare any data for the frame after the run.
