    Class SystemManager

    Entry point for systems. Allow to register systems.

    Inherits from

    • Object (base class)

    Constructors

    Name Description
    this

    Properties

    Name Type Description
    statAll [get] const(Stat) Reference on profiling statistics of all the system's prepare, unprepare and run.
    statEnabled [get] bool Tells whether statistics profiling is enabled or not.
    statRate [get, set] const(core.time.Duration) Set/get rate at which StatEvent events are sent.
    statRun [get] const(Stat) Reference on profiling statistics of all the system's run.

    Methods

    Name Description
    disableStat Disable statistics profiling on the system-manager and all its registered systems.
    enableStat Enable statistics profiling on the system-manager and all its registered systems. A delegate dg can be given, the rate at which it will be called to provide significant stat's.
    opSlice Return a bidirectional range on the list of the registered systems.
    prepare Prepare all the registered systems.
    register Register a new system.
    run Run all the registered systems.
    runFull Prepare, run and unprepare all the registered systems.
    unprepare Unprepare all the registered systems.
    unregister Unregister a system.
