    Function EventManager.emit

    Emit an event.

    It will be dispatched to all receivers that subscribed to it.

    Prototypes

    void emit(E)(
      in ref E event
    )
    if (isEvent!E);

    void emit(E, Args...)(
      auto ref Args args
    )
    if (isEvent!E);

    Example

    @event struct MyEvent
{
    int data;
}

auto em = new EventManager;

auto e = MyEvent(43);

em.emit(e);
em.emit!MyEvent(42);
