    Function Entity.register

    Register a component C to an entity.

    Prototype

    C* register(C, Args...)(
      Args args
    )
    if (isComponent!C);

    Parameters

    NameDescription
    C Component to register.
    args List of arguments to initialize the component, will be passed to its constructor. Optional.

    Returns

    A pointer on the component for this entity.

    Throws

    EntityException if the entity is invalid. ComponentException if there is no room for that component or if the component is already registered.
